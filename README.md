# Art Store

Art Store is an e-commerce website created as the final project for a Front-end Development course.  
It is an online store for stationery items, where users can browse products, sign up and log in, add items to the cart and place orders.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/tsuki-koi/art-store.git
   ```

2. Go into the project folder that you cloned:
   ```bash
   cd art-store
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```
   
4. Start the project:
   ```bash
   npm start
   ```
